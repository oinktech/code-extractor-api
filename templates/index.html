API 使用教程

請參照以下代碼將此 API 集成到靜態網站：


fetch('https://code-extractor-api.onrender.com/extract', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify({
        code: '<h1>Example</h1><p>This is a paragraph</p>',
        tags: ['h1', 'p']
    }),
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));

API 回應格式


{
    "tag": "h1",
    "text": "Example",
    "placeholder": "無 placeholder"
}
