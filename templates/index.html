API 使用教程

要在您的靜態網站中使用此 API，請按照以下步驟操作：

1. 引入 jQuery

在您的 HTML 文件中引入 jQuery：

<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

2. 使用 Fetch API 發送請求

以下是如何獲取標籤的範例：

$.ajax({
    url: 'https://your-api-url/get_tags', // 請替換為實際 API 地址
    type: 'POST',
    contentType: 'application/json',
    data: JSON.stringify({ code: '
示例標題
這是一段文字。
' }),
    success: function(response) {
        console.log(response);
    },
    error: function(error) {
        console.error('錯誤:', error);
    }
});

3. 提取標籤中的文本

以下是如何提取標籤中的文本的範例：

$.ajax({
    url: 'https://your-api-url/extract', // 請替換為實際 API 地址
    type: 'POST',
    contentType: 'application/json',
    data: JSON.stringify({
        code: '<h1>示例標題</h1><p>這是一段文字。</p>',
        tags: ['h1', 'p']
    }),
    success: function(response) {
        console.log(response);
    },
    error: function(error) {
        console.error('錯誤:', error);
    }
});

這些示例會將標籤及其文本輸出到控制台。
