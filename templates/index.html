API 使用教程

此 API 允許您提取 HTML 代碼中的標籤和文本。以下是如何使用它：

1. 獲取標籤

發送 POST 請求到 /get_tags，內容為 HTML 程式碼。

POST /get_tags
Content-Type: application/json

{
    "code": "<h1>示例標題</h1><p>這是一段文字。</p>"
}

示例輸出：

{
    "tags": ["h1", "p"]
}

2. 提取文字和標籤

發送 POST 請求到 /extract，內容為 HTML 程式碼和選擇的標籤。

POST /extract
Content-Type: application/json

{
    "code": "<h1>示例標題</h1><p>這是一段文字。</p>",
    "tags": ["h1", "p"]
}

示例輸出：

[
    {
        "tag": "h1",
        "text": "示例標題",
        "placeholder": "無placeholder"
    },
    {
        "tag": "p",
        "text": "這是一段文字。",
        "placeholder": "無placeholder"
    }
]
